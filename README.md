# Socket.io with Typescript

## Install & Run

Will run both client and server in parallel

1. `npm install`
2. `npm start`

### Installation in two tabs

Open two terminals and run the following commands

1. `npm run install`
2. `npm run server-install`

### Run project in two tabs

Open two terminals and run the following commands

1. `npm run client`
2. `npm run server`
